##### Clash机场 / 节点订阅简单使用说明

目前大多数翻墙机场都推荐通过 Clash机场 + 节点订阅 的方式使用，配置简单且稳定。

基本流程：

在机场官网复制 Clash 订阅链接
安装 Clash for Windows、Clash Verge、Shadowrocket、Stash 等客户端
在客户端中导入订阅并更新节点
选择规则模式（Rule），连接低延迟节点即可
这种方式可以自动同步机场节点，是目前较为省心的外网梯子使用方案。
